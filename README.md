# A2sv-blog

this is the simple blog post assessment



## Installation

Install the project with npm in you local pc

```bash
  git clone https://github.com/chalaa/a2sv-blog/

  npm install

  npm run dev
```
Or directly access the up by singing up on the 

```
https://a2sv-blog.vercel.app/
```
